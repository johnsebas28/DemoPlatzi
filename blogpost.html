Estoy aprendiendo git

Aquí voy a colocar algunos comandos que he aprendido.

  • git init: Inicializa una carpeta como repositorio git.
  • git add: Envía cambios a la zona de staging
  • git commit: confirma los cambios en el repositorio.
  • git log: Para mirar los commits que ha tenido un archivo
  • git status: Muestra el estado actual de los archivos.
  • git rm --cached: Elimina el archivo del área de staging y del próximo commit pero los mantienen en el disco duro.
  • git rm --force: Elimina los archivos de git y del disco duro.
